Programming One

Welcome to programming one. To get a computer to do a job, you need to give it instructions in a language that it can understand. Computers are able to understand, a range of languages that have more limited vocabulary and stricter rules than the natural languages that humans speak. These are known as programming languages. The language you will be learning to use in this module is called Java.
