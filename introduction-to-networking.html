Introduction to Networking

Welcome to the Introduction to Network Technology module. In this module you will learn the fundamentals of computer networking and the impact that global communications has on the modern world. Using standard models such as the OSI and TCP/IP models, you will investigate the technologies mapped to each layer and the interactions between these technologies at each layer.
